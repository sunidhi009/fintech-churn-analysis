# Fintech User Retention & Churn Analysis

## Objective
Analyze customer churn patterns and identify retention opportunities for a fintech platform.

## Dataset
Bank Customer Churn Dataset containing 10,000+ user records.

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  

## Analysis Performed
• Customer churn distribution  
• User activation funnel  
• Cohort churn analysis  
• Correlation heatmap  

## Key Insights
- Users aged 46–55 show the highest churn rate.
- Active members churn less than inactive users.
- Customers with multiple products are more likely to stay.
